## CUDA Image Encoder

A high-performance CUDA image conversion pipeline supporting RAW and DICOM images with GPU filtering, custom GPU compression, asynchronous execution, parallel pipelines, and optimized memory management.

**Highlights**

* CUDA C++
* Custom GPU Deflate implementation
* Producer–Consumer architecture
* Multi-threaded pipeline
* Performance telemetry
* Batch processing
* Medical image support

An AI-powered conversational assistant built with FastAPI, LangGraph, and LLMs featuring streaming responses, tool calling, session memory, and astrology-based reasoning.

**Tech**

* LangGraph
* LangChain
* LLMs
* SSE Streaming
